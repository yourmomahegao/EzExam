----------------------------------------------
----------------- Инструкция -----------------
----------------------------------------------

1. Как добавить изображение?
Поместите требуемые изображение в папку с EzExam.

2. Как управлять приложением?
Стрелка вверх - Увеличить прозрачность;
Стрелка вниз - Уменьшить прозрачность;
Стрелка вправо - Следующее изображение;
Стрелка влево - Изменить позицию изображения.

3. Приложение не управляется?
Возможно оно не в фокусе, нажмите на окно с приложением.
